눈치싸움 개인정보처리방침

Privacy Policy · 최종 수정일: 2026년 5월 16일

눈치싸움은 개인정보를 수집하지 않습니다. 사용자 식별, 계정 생성, 위치 추적 등 일체의 개인정보 처리를 하지 않습니다.

1. 개요

주식회사 서동메디칼(이하 "회사")이 제공하는 눈치싸움 앱 (패키지명: com.sdongworks.vibecheck)은 도로 CCTV 이미지를 분석하여 주변 혼잡도를 알려주는 서비스입니다.

본 방침은 앱이 처리하는 정보의 범위와 목적을 투명하게 안내하기 위해 작성되었습니다.

2. 수집하는 정보

회사는 사용자 개인을 식별할 수 있는 어떠한 정보도 수집·저장하지 않습니다.

  • 이름, 이메일, 전화번호 등 연락처 정보 — 수집 안 함
  • GPS 위치 정보 — 수집 안 함 (장소명 텍스트 검색만 사용)
  • 기기 고유 식별자 — 수집 안 함
  • 얼굴·생체 정보 — 수집 안 함

3. CCTV 이미지 처리

앱은 국가교통정보센터(ITS) 및 경기도교통정보센터(GITS)가 공개 제공하는 도로 CCTV 스냅샷 이미지를 실시간으로 불러와 화면에 표시합니다.

  • 이미지는 서버에 저장되지 않으며, 화면 표시 후 즉시 메모리에서 해제됩니다.
  • AI 분석은 차량(자동차·버스·트럭·오토바이)만 감지하며, 사람·얼굴 등 개인 식별 클래스는 사용하지 않습니다.
  • 분석 결과(차량 대수)는 기기 내 메모리에만 존재하며 외부 서버로 전송되지 않습니다.

4. 광고 (Google AdMob)

앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 광고 최적화 목적으로 광고 ID(GAID) 등 제한적인 정보를 수집할 수 있으며, 이는 Google의 개인정보처리방침에 따라 처리됩니다.

  • Google 개인정보처리방침: https://policies.google.com/privacy
  • 광고 ID 재설정 및 개인 맞춤 광고 거부: 기기 설정 → Google → 광고에서 변경 가능합니다.

5. 제3자 API 서비스

앱은 다음 외부 API를 사용합니다. 각 서비스의 개인정보처리방침이 적용됩니다.

  • 카카오 Local API — 장소명 검색 (카카오 개인정보처리방침 적용)
  • 국가교통정보센터(ITS) CCTV API — 도로 CCTV 스냅샷 제공
  • 경기도교통정보센터(GITS) CCTV API — 경기도 도로 CCTV 스냅샷 제공

6. 아동 개인정보

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 고의로 수집하지 않습니다.

7. 방침 변경

본 방침이 변경될 경우 앱 업데이트 또는 본 페이지를 통해 사전 고지합니다. 중요한 변경 사항은 앱 내 공지로 안내합니다.
